# Changelog

## [Unreleased]

### 1.1.0 — 看板 · 子任务 · 标签 · 任务详情

**看板视图**
- 三列看板（待办 / 进行中 / 已完成）
- 列表 / 看板一键切换
- 看板卡片点击进入任务详情
- 下拉菜单快速移动任务状态

**子任务**
- 一层嵌套子任务，父任务下折叠展示
- 父任务标记完成 → 级联完成所有子任务
- 子任务独立支持状态切换、标签、负责人
- 任务详情页管理子任务（添加 / 删除 / 状态切换）

**标签系统**
- 清单级别标签（Tag + TaskTag 多对多）
- 创建 / 编辑任务时内联添加标签（回车或按钮）
- 任务卡片和详情页展示标签
- 工具栏标签下拉筛选

**任务详情页**
- 点击任务卡片进入独立详情页
- 编辑模式：标题、状态、优先级、截止日期、负责人、描述、标签
- 子任务列表管理
- 删除任务（确认弹窗）
- 截止日期为空显示「无」

**权限优化**
- 前端 UI 门控：编辑/删除按钮仅创建者和管理者可见
- 任务详情页根据用户角色展示对应操作
- 子任务删除仅管理员可操作
- 参与者可切换自己任务状态、添加子任务

**UI 修复**
- 文本长度限制：标题 60 / 标签 30 / 清单名 50 字符
- 全量 `overflow-hidden` + `break-words` 防止文本溢出
- 标签筛选按钮长文本省略号截断
- 编辑对话框回车键仅添加标签，不触发表单提交
- 成员搜索排除已有成员
- 成员添加改为 userID 精确匹配
- 排序下拉暗色模式可见性修复
- 清单描述支持展开/收起
- 创建者可见删除清单按钮

### 1.0.0 — 团队待办清单 MVP

**基础设施**
- Next.js 15 App Router + React Server Components
- TypeScript 严格模式
- Tailwind CSS + shadcn/ui 组件库
- PostgreSQL + Prisma ORM 数据持久化
- NextAuth.js v5 邮箱密码认证
- Docker Compose 一键部署
- pnpm 包管理

**账号与认证**
- 邮箱 + 密码注册与登录
- NextAuth.js Session 管理
- CSRF 保护

**清单管理**
- 创建清单（成员自动成为创建者）
- 编辑清单名称与描述
- 删除清单（仅创建者）

**权限系统**
- 三级角色：创建者 / 管理者 / 参与者
- 应用层中间件校验（Next.js Middleware + API Route Guards）
- 创建者可转让创建者角色、修改成员角色
- 管理者可添加/移除成员、管理任务
- 参与者仅操作自己负责的任务
- 无权限操作统一返回 403

**任务管理**
- 任务 CRUD
- 固定三态：待办 → 进行中 → 已完成
- 优先级：高 / 中 / 低
- 截止日期
- 指派负责人
- 排序：优先级 / 截止日期 / 创建时间
- 筛选：按状态、按指派人、「只看我的」
- 标题关键词搜索

**成员管理**
- 组合搜索（用户名 / 显示名 / 邮箱）添加成员
- 实时成员列表

**UI / UX**
- 响应式布局（桌面 / 平板 / 手机）
- 仪表盘（所有参与清单概览）
- 清单详情页（任务列表 + 成员面板）
- 暗色模式
