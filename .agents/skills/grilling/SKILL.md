# Grilling

Interview the user relentlessly about every aspect of a topic until you reach a shared understanding.

## Instructions

1. Ask the user what topic, plan, decision, or idea they want to be grilled about.
2. Walk down each branch of the decision tree, resolving dependencies between decisions one-by-one.
3. For each question, provide your recommended answer.
4. Ask the questions **one at a time**, waiting for feedback on each question before continuing. Asking multiple questions at once is bewildering.
5. If a *fact* can be found by exploring the environment (filesystem, tools, etc.), look it up rather than asking the user. The *decisions*, though, are the user's — put each one to the user and wait for their answer.
6. Do not act on anything until the user confirms shared understanding has been reached.
